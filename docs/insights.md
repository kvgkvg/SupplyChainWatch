# Verified Insight Targets

These are the six report-grade insights the project is designed to verify. When live data is available, use the methods below to replace the current hypothesis status with measured values, charts, and screenshots.

## 1. Chokepoint Disruption Ripple Effect

**Hypothesis:** A material drop in Suez vessel throughput is followed by higher container freight rates.

**Evidence to collect:**
- `chokepoint_status.vessel_count` for Suez Canal.
- `freight_indices.value` for `FBX_GLOBAL` or `WCI_GLOBAL`.
- Cross-correlation over 90-180 days.

**Method:** Align daily series, compute raw and detrended correlation, then test lag windows from -30 to +30 days.

**Expected output:** Story Mode narrative plus lag/correlation chart.

## 2. Shanghai Port Lead-Lag Relationship

**Hypothesis:** Shanghai congestion leads freight-rate pressure with a short lag.

**Evidence to collect:**
- `port_congestion.total_in_area` for Shanghai.
- `FBX_GLOBAL` or route-specific FBX proxy.
- High-z-score congestion anomalies.

**Method:** Cross-correlation and event study around congestion spikes.

**Expected output:** Port congestion chart, freight index overlay, and LLM narrative with guarded causality.

## 3. Bunker Prices as Cost Pressure Signal

**Hypothesis:** Bunker price increases coincide with freight-rate movement and can strengthen explanations of rate changes.

**Evidence to collect:**
- `bunker_prices.price_usd_per_ton`.
- `freight_indices.value`.
- Forecast commentary MAPE and confidence interval width.

**Method:** Compare 7-day and 30-day changes, then include bunker context in forecast commentary.

**Expected output:** Forecast card with commentary that explicitly states MAPE.

## 4. Port Congestion Anomaly Concentration

**Hypothesis:** High-severity port anomalies cluster around major transshipment and gateway ports.

**Evidence to collect:**
- `anomalies` where `entity_type='port'`.
- Port reference metadata from `ports`.
- Congestion timelines.

**Method:** Count high-severity events by port and region, then inspect outlier timelines.

**Expected output:** Anomaly timeline and expandable AI explanation.

## 5. Freight Index Co-Movement

**Hypothesis:** Container indices co-move more strongly with each other than with dry-bulk indices.

**Evidence to collect:**
- `BDI`, `FBX_GLOBAL`, `WCI_GLOBAL`, optional `SCFI`.
- Correlation matrix over 90/180/365-day windows.

**Method:** Pearson correlation on aligned daily data; repeat with detrending for robustness.

**Expected output:** Correlation heatmap in Insights Hub.

## 6. Forecast Reliability and Directional Use

**Hypothesis:** Short-horizon forecasts are useful directionally but should be shown with explicit uncertainty.

**Evidence to collect:**
- `forecasts.predictions`.
- `forecasts.metrics.mape`.
- Forecast commentary generated by the LLM.

**Method:** Backtest hold-out windows, compute MAPE, and compare prediction bands to observed values when available.

**Expected output:** 14-day forecast chart with confidence interval and commentary.
